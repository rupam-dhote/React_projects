## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/rupam-dhote/React_projects.git
```

Navigate to any project folder:

```bash
cd React_projects/project-folder
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
